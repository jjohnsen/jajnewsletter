= jajNewsletter

Newsletter extension for eZ Publish

=== System Requirements

* eZ Publish 3.10 or 4.2
* premailer

== Installation

=== Extension

=== Premailer

Get and install latest premailer from http://code.google.com/p/premailer/wiki/InstallingPremailer

   wget http://code.dunae.ca/premailer.web/gems/premailer-1.5.0.gem
   sudo gem install premailer-1.5.0.gem
